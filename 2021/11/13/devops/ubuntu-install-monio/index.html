ubuntu安装minio,用作个人的oss服务

minio是什么?可用作文件对象存储服务。

MinIO 是在 GNU Affero 通用公共许可证 v3.0 下发布的高性能对象存储。 它是与 Amazon S3 云存储服务兼容的 API。 使用 MinIO 为机器学习、分析和应用程序数据工作负载构建高性能基础架构。官网介绍

用docker-compose安装minio

本次安装使用的是docker-compose，省去了敲命令的烦恼，同时要是迁移服务器的话，只需要把docker-compose文件切到另一台服务器上面执行便可以。

docker-compose使用步骤

  • 安装docker服务
    • apt install docker
  • 安装docker-compose服务
    • apt install docker-compose
  • 常用的docker-compose命令
    • docker-compose up -d
    • docker-compose down -d
